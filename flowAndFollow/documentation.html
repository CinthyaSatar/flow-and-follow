Flow and Follow - Period Tracking Application

About

Flow and Follow is a user-friendly period tracking application that helps users monitor their menstrual cycle. Features include cycle prediction, symptom tracking, and personal health insights.

Setup Instructions

Prerequisites

  • Python 3.x installed on your computer
  • Command line/terminal access

Installation Steps

  1. Navigate to the project directory: cd projects/actual_project/webapp
  2. Create a virtual environment: python -m venv venv
  3. Activate the virtual environment:
    Windows: .\venv\Scripts\activate
    Mac/Linux: source venv/bin/activate
  4. Install required packages: pip install flask flask-wtf flask-sqlalchemy flask-bcrypt flask-login flask-migrate pillow email_validator
  5. Run the application: python run.py
  6. Open your web browser and visit: http://127.0.0.1:5000

Features

  • User Registration and Login
  • Period Cycle Tracking
  • Symptom Recording
  • Personal Health Insights
  • Cycle Predictions

Technical Stack

  • Backend: Flask (Python)
  • Database: SQLite
  • Authentication: Flask-Login
  • Forms: Flask-WTF
  • Password Hashing: Flask-Bcrypt
